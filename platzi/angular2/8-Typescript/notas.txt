babel vs typescript

la diferencia entre babel y typescript es que babel 
sólo permite transpilar de ES6 a ES5 
(a no ser que uses algún módulo o lo combines mediante GulpJS).

TypeScript permite compilar de TypeScript a ES6 o ES5, 
con lo que ya llevaría 
implícita la conversión de ES6 a ES5 si el destino está en ES5.

Documentación: https://www.typescriptlang.org/docs/tutorial.html

se puede usar gulp para automatizar la tarea de compilacion

Le recomiendo usarlo con GulpJS
-lo único que necesitas hacer generar un package.json(npm init -y)
-después npm install --save-dev gulp
-después npm install --save-dev gulp-typescript
-creas un archivo gulpfile.js
y creas tus gulp.task(tareas), 
en lo personal es mejor trabajarlo así.
