------ Inicio clase

Typings, nos permite tener definiciones de librerías de terceros.
Para instalarlo usamos npm install typings -g
Para obtener la referencia a una librería usamos 
typings install dt~jquery --global --save

definicio de librerias de terceros dentro de typescript
por ejemplo jquery
